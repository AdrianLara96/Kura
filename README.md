# 🎨 Kura

> Descubre el arte. Crea tu legado.

Plataforma de descubrimiento cultural que conecta obras maestras de museos internacionales con colecciones temáticas creadas por usuarios.

---

## 🛠️ Stack

- **Frontend:** Vue 3 + Vite + JavaScript
- **State:** Pinia
- **Router:** Vue Router 4
- **Backend:** Supabase (Auth + DB + Storage)
- **APIs:** Rijksmuseum, The Met, Museo del Prado

---

## ⚙️ Instalación

```bash
npm install
npm run dev
